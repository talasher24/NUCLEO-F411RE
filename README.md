# Project Title
Working with NUCLEO-F411RE

## Description
The project executes the following commands received through serial terminal:
```
ping
version
pwm_start
pwm_dc XXX
pwm_stop
crc_whole_flash_calc
iwdg_test
flash_lock
set_SN
get_SN
start_tick
stop_tick
flash_lock
assert_0
clear_assert_flag
```

## Topics included in the project:
Serial protocols:
```
I2C
UART
```
ST MCU peripherals:
```
DMA
Timers
RTC
CRC
Internal flash
Watchdog
```
MEMS sensors:
```
LSM6DSL
```

### Prerequisites
What things you need to install the software and how to install them
```
STM32CubeMX
STM32CubeIDE
STM32 ST-LINK Utility
Serial communication terminal 
```
